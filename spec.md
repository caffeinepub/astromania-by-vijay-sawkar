# Specification

## Summary
**Goal:** Change the logo background in the Header component from a white/mosaic style to a bluish color that fits the dark cosmic theme.

**Planned changes:**
- Update the Header component's logo area background from white/mosaic to a deep or medium blue tone (e.g., `#1a3a6b` or a subtle cosmic blue gradient)
- Ensure the change is consistent across desktop and mobile views

**User-visible outcome:** The logo in the header now displays against a bluish background instead of the previous white/mosaic pattern, harmonizing with the app's dark cosmic theme.
